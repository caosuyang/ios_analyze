# ios_analyze

## 为什么要建这个仓库

- 以文章形式记录平时开发遇到的难点。
- 主要包括：iOS平时遇到的知识点，iOS底层原理分析，iOS开源框架源代码解析，性能监控和优化线上线下解决方案等。
- 我会不断更新这个仓库中的文章，如果想要关注可以点 `star`。
